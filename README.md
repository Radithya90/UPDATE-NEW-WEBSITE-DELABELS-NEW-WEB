# Website Kelas Kami

Website sederhana untuk komunitas kelas 8.11 MTSN 1 Kota Bekasi.

## Fitur

- Halaman beranda dengan informasi kelas
- Daftar siswa dan informasi guru
- Jadwal pelajaran
- Galeri kegiatan
- Informasi kontak

## Cara Menjalankan

1. Buka file `index.html` di browser
2. Atau gunakan server lokal:
   ```bash
   python -m http.server 8000
   ```
   Kemudian buka `http://localhost:8000`

## Teknologi

- HTML5
- CSS3
- JavaScript
